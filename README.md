```
Buenas tardes Jorge;

Estos son los detalles de la prueba técnica para el puesto de desarrollador Front-end en Angular:

Como recomendación inicial puedes ayudarte de Angular Material para facilitar la estética y funcionalidad.

1º - Debes consumir los datos de esta API de películas series y videojuegos: https://www.omdbapi.com/ para ello debes registrarte para que te faciliten un “Apikey” y poder realizar búsquedas y obtener datos.

2º- Tendrás que maquetar y desarrollar una pequeña aplicación con un buscador que muestre los resultados de las búsquedas en una tabla.
IMPORTANTE mirar la documentación y hacer la búsqueda “By Search” de la API para obtener más de un solo resultado.

3º- Después de realizar cualquier búsqueda y mostrar los resultados en la tabla deberíamos tener la posibilidad de guardar localmente los resultados que nos gusten como favoritos. La página de favoritos debe mostrar los resultados en formato TARJETA o CARTA cuidando el responsive y la estética.

4º- Crear una barra de navegación para movernos entre los favoritos y la tabla de búsquedas.

5º- Estando en la página de favoritos tenemos que poder filtrar nuestra lista de favoritos por varios campos simultáneamente si quisiéramos, es decir añadir los inputs necesarios para cada campo adicional que queramos filtrar.

6º- Nuestros favoritos deberían también permitirnos editar un campo nuevo que debes crear para poner una descripción/valoración de la película, serie o videojuego que tengamos en nuestra lista, que se guarde localmente y se pueda cambiar individualmente entre el modo edición y el modo vista a través de un botón.

7º- Por último, deberíamos poder eliminar un elemento de la lista de favoritos.

RECUERDA: Debes compartir el proyecto en Github y darnos acceso a mdelgado@controlli.es , ejimenez@controlli.es y jbacarisas@controlli.es .

Es importante que seas riguroso con los puntos principales de la prueba. Siéntete libre posteriormente de estilizar, añadir elementos y funcionalidades como consideres. Todo suma pero prioriza los puntos principales de la prueba.

Valoraremos la estructura del proyecto, organización, código limpio y escalable, estilos y estética, también el uso correcto de GIT.

Muchos ánimos y quedamos a la espera. Cualquier consulta no dudes en contactar.

Saludos.
```
